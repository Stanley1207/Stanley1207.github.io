About the Engine

EchoPad is a lightweight, modern 2D game engine built in C++ for rapid prototyping and experimentation. Designed for responsiveness and modularity, EchoPad now supports full controller integration for immersive input handling.

Demo Video

Technologies Used

  • C++
  • Lua & LuaBridge (Support flexible game logic)
  • GLM (OpenGL Mathematics)
  • Rapidjson (Parse and generate JSON)
  • SDL2 (Simple DirectMedia Layer)
  • Box2D (Physics Engine)
